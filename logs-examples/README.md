# logs-examples

本目录不保存完整个人日志，避免泄露账户、路径、项目名、token、会话信息或 Chrome 用户数据。

只保存脱敏后的关键证据，例如：

```text
bundled_executable_relocation_failed
node-repl-missing
missingHelperPath=true
browser_use_setup_failed
codex-windows-sandbox-setup.exe missing
codex-command-runner.exe missing
SUCCESS; SAFE_TO_REOPEN_CODEX
```

本次案例关键结果：

```text
Status: repaired
PackageVersion: 26.616.9593.0
CodexFingerprint: 38dff8711e296435
RgFingerprint: ada252862d154cdd
CuaFingerprint: 1b23c930bdf84ed6
Chrome 成功打开 openai.com
页面标题：OpenAI | Research & Deployment
```

隐私处理规则：

```text
不上传完整截图
不上传完整日志
不上传邮箱、token、Cookie、会话文件
不暴露真实 Windows 用户名
本机路径统一写成 %USERPROFILE% 或 %LOCALAPPDATA%
```
